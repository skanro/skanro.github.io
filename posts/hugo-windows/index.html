hugo windows10

hugo 다운로드

hugo 를 윈도우에서 사용하기 위한 파일들이다. hugo release 링크에서 hugo windows 최신버전을 다운받는다.

압축을 풀고 폴더에 저장하고 경로를 복사한다. 버전업데이트시 hugo.exe 파일만 바꾸면 된다.
