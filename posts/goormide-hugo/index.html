구름IDE 에서 hugo server

2021년 01월 22일

hugo 구름ide

hugo server 는 Hugo 사이트를 구축하는 자체 웹서버다. 사이트를 수정하고 변경사항을 미리볼때 자주 사용하는데 구름ide 에서 사용하게 되면 웹 서버가 열리지 않는다.

해결방법

nodejs로 컨테이너 생성시

hugo server -bind='0.0.0.0' -p 3000

상단 메뉴 프로젝트 > 실행 URL 과 포트에서 웹서버 주소를 알 수 있다. 포트랑 -p 3000 이랑 맞추면 된다.
