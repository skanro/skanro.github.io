Hugo 이미지 캡션 넣기

2021년 02월 12일

hugo 이미지 캡션

마크다운 이미지 캡션

hugo 블로그에서 마크다운으로 이미지 타이틀 속성을 입력하면 이미지캡션을 보여주는 방법이다.

![텍스트](이미지주소 "타이틀")

위와 같이 이미지에 타이틀 속성을 넣을 수 있다.

render_image.html

{{ if .Title }}
  <figure>
    <img src="{{ .Destination | safeURL }}" alt="{{ .Text }}">
    <figcaption>{{ .Title }}</figcaption>
  </figure>
{{ else }}
  <img src="{{ .Destination | safeURL }}" alt="{{ .Text }}">
{{ end }}

layouts\_default\_markup\render-image.html 경로에 위 내용으로 파일을 생성한다.
