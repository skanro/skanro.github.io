정적사이트에서 파일/이미지 업로드하기

2021년 01월 30일

github hugo

위 방법처럼 업로드하면 마크다운으로 변환된 이미지를 복사해서 바로 붙여넣기 할 수 있다.

어떤 깃허브 저장소 이슈라도 상관없지만 블로그에 업로드할겸 hugo 프로젝트 저장소에 이슈를 올려 글을 남긴다. 이슈를 올리지않고 지워도 업로드된 이미지가 사라지지는 않는다.

파일 업로드 최대 용량은 25MB 이고 이미지 용량은 10MB 이다. 네이버와 티스토리 블로그 업로드 최대 용량은 10MB.

  • 이미지 (.png, .gif, .jpg)
  • 로그 파일 (.log)
  • 워드 파일, 파워포인트 파일, 엑셀 문서 (.docx, .pptx, .xlsx)
  • 텍스트 파일 (.txt)
  • PDF (.pdf)
  • 압축 파일 (.zip , .gz)

위는 업로드 가능한 파일 형식들이다.
