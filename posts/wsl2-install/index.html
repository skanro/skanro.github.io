WSL2(Windows Subsystem for Linux2) 설치하기

2021년 02월 20일

windows10 wsl linux

설치하기 전

  • x64 시스템의 경우: 버전 1903 이상, 빌드 18362 이상
    ARM64 시스템의 경우: 버전 2004 이상, 빌드 19041 이상
    (버전 및 빌드 번호 확인 Windows 로고 키 + R 선택, winver 실행)
  • CPU 가상화 지원 (작업관리자 성능탭에서 사용확인, Bios 에서 설정)
  • 관리자권한으로 powershell 실행 필수

https://docs.microsoft.com/ko-kr/windows/wsl/install-win10

Windows 10 WSL2 설치하기

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

Linux용 Windows 하위 시스템 옵션 기능 사용 설정.

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

Virtual Machine 플랫폼 옵션 기능 사용 설정

마이크로소프트 스토어에서 우분투 설치하고 다시시작.

우분투 사용자 이름 비밀번호 설정.

wsl --set-default-version 2
wsl --list --verbose
wsl --set-version Ubuntu 2

WSL2 를 기본버전으로 설정, Linux 배포에 할당된 WSL 버전 확인, WSL 배포 버전 설정

커널 업데이트가 필요할 시 링크 에서 최신패키지 다운로드
